# Projekt Data Mining

Projekt zaliczeniowy z przedmiotu Data Mining. Całościowa analiza problemu z wykorzystaniem metod data mining oraz budowa modeli predykcyjnych.

## Wymagania projektu

- **Modele**: drzewa decyzyjne, modele zespołowe (ensemble), sieci neuronowe
- **Metodologia**: CRISP-DM lub SEMMA
- **Długość**: ~30 stron (czcionka 12)
- **Format**: Word (.docx) + PDF
- **Zawartość**: spis treści, wstęp (cel, zakres, źródło danych), opis zbioru, analiza, modelowanie, wnioski, kody w załączniku
- **Wszystkie wyniki** muszą mieć komentarze
- **Link do danych** musi być w projekcie

## Struktura

```
DM_project/
├── README.md
├── data/              # Zbiory danych
├── notebooks/         # Jupyter notebooks z analizą
├── src/              # Kody źródłowe
└── report/           # Dokument projektu
```
